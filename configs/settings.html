Editor CSS
Campo exclusivo para desenvolvedores: Este editor de CSS permite realizar personalizações avançadas e ajustes visuais específicos na loja. Utilize-o para modificar estilos, cores, tamanhos e qualquer outro detalhe visual necessário.
Recomenda-se que apenas usuários com conhecimentos em CSS utilizem este recurso para garantir que as alterações sejam feitas corretamente.
